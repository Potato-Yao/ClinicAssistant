About The ClinicAssistant

When repairing laptops in the computer clinic, we often need to use multiple tools to check laptop's state, change system settings, perform stress test and so on, which is not convenient to switching between different tools.
Thus, I developed this program, to make most common used operations can be done by "one click".
Thank you for using ClinicAssistant!
